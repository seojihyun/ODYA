# 파인애플 소개
위치기반 AR을 사용한 위치추적 어플인 '파인애플'은 단체여행시 여행객이 가이드를 놓쳐 무리에서 벗어난 경우 가이드의 위치, 다른 여행객들의 위치를 안내 해주기 위한 서비스 입니다.  GPS, AR 기능을 이용하여 현재 사용자 위치에 대한 상대방의 상대적 위치를 더 쉽게 파악할 수 있습니다.
